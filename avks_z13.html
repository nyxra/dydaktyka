Kaskadowe Arkusze Stylów CSS

Cechy CSS

CSS - Kaskadowe Arkusze Stylów (Cascading Style Sheets) to język służący do opisu wyglądu dokumentów HTML.

Style pozwalają oddzielić strukturę dokumentu od jego prezentacji.

Dzięki zewnętrznym arkuszom stylów możemy zarządzać wyglądem wielu stron jednocześnie.

Zalety stosowania CSS

Modyfikacja wyglądu strony jest szybka i wygodna.

Dokumenty są bardziej przejrzyste i krótsze.
