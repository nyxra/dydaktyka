1. Składnia teoretyczna

CSS składa się z selektora oraz deklaracji. Deklaracja zawiera właściwości i jej wartość, np. colo: red; Styl może być otoczony ramką(np.tabela).

2. Gdzie wstawiany jest styl zewnętzny

Styl zewnętzny wstawia się w sekcji head dokumentu HTML za pomocą znacznika link. Może występować wiele razy w dokumencie.

3. Najprostrza forma wstawiania stylu

Najprostrza forma to uzycie znacznika link z atrybutem rel="stylesheet" oraz href wskazującym plik CSS.

4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasad musi być zachowana przy większej ilości dołączanych styli?

Można dolądzyć wiele arkuszy stylów, ale należy zachować kolejność, ponieważ ostatni sytl może nadpisywać poprzedni.

5. Konflikty stylów

Konflikty występują, gdy te same elementy mają rożne style. Decyduje zasada kaskady - wazniejszy jest styl bardziej szczegółowy lub póżniejszy

6. Tworzenie plików CSS w edytorze tekstowym

Plik CSS można utworzyć w zwykłym edytorze tekstowym, zapisując go w roszrzeniem .css i wpisując reguly stylów.
