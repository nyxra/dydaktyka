Rodzaje selektorów CSS

Selektory CSS

  • Elementu – p { }
  • Klasy – .klasa { }
  • ID – #id { }
  • Uniwersalny – * { }
  • Potomka – div p { }
  • Dziecka – div > p { }
  • Atrybutu – a[href] { }
  • Pseudoklasy – a:hover { }
  • Pseudoelementu – p::before { }
  • Grupowania – h1, h2 { }

Zagnieżdżone wyliczenia (3 poziomy)

width: calc(100% - 2rem)
padding: calc( calc(100% / 20) - 4px )
font-size: calc( calc(0.5rem + 1vw) - calc(0.1rem) )
